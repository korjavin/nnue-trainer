# Setup New Dev Host Skill

This directory contains the Claude skill instructions for bootstrapping and configuring a new development host.

## Files
- `SKILL.md` - Core instructions, guidelines, and commands for the AI agent to set up a new host.

## Usage
When setting up a new machine or environment, Claude Code will activate this skill to follow detailed, clean, and version-safe installation steps for tools like `npm`, `claude`, `codex`, `agy`, `ripgrep`, `gh`, `nvim`, `golang`, and `claude-common-skills`.
