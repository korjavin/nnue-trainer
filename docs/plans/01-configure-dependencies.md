# Configure Maven Dependencies for WebSocket and JSON

Implement the Maven dependency configuration in `pom.xml` to include websocket client and JSON parsing libraries.

## Tasks

- [ ] Task 1: Update pom.xml dependencies

### Task 1: Update pom.xml dependencies
1. Add `org.java-websocket:Java-WebSocket:1.5.7` (or similar Java WebSocket client library) to `pom.xml`.
2. Add `com.fasterxml.jackson.core:jackson-databind` for JSON serializing and deserializing.
3. Ensure the project builds successfully with `mvn clean compile`.\n
